    GoVWA (Go Vulnerable Web Application)

    GoVWA adalah aplikasi web yang dikembangkan untuk membantu pentester dan programmer untuk mempelajari kerentanan yang sering terjadi dalam aplikasi web yang dikembangkan menggunakan golang. Kerentanan yang ada di GoVWA adalah kerentanan yang paling umum ditemukan dalam aplikasi web saat ini. Jadi ini akan membantu programmer mengenali kerentanan sebelum terjadi pada aplikasi mereka. Govwa juga bisa menjadi aplikasi tambahan dari laboratorium pentest Anda untuk belajar dan mengajar. GoVWA bisa di unduh di link berikut : GoVWA

    NVWA (NemoSec Vulnerable Web Application)

    NVWA adalah aplikasi web yang dikembangkan untuk membantu pentester dan programmer untuk mempelajari kerentanan yang sering terjadi dalam aplikasi web yang dikembangkan menggunakan PHP.
